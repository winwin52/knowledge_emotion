# knowledge_emotion

> 2026 中国大学生计算机设计大赛 — 面向运动康复的智能情感交互系统设计

## 项目简介

本仓库是中国大学生计算机设计大赛（生物医学工程创新设计竞赛）参赛项目的知识库与研究笔记，聚焦于**运动康复场景下的智能情感交互系统**设计。

核心问题是：运动康复过程周期漫长、训练枯燥，导致患者依从性低、训练中断率高。本项目探索如何利用**情感计算、多模态感知、人机交互**技术，为运动康复提供情感支持与智能引导。

## 核心技术方向

| 方向 | 说明 |
|------|------|
| 多模态情感识别 | 融合面部表情、语音、生理信号（EEG/ECG/EDA）、雷达等多种模态 |
| 非接触式感知 | 毫米波雷达心率呼吸监测、摄像头面部识别 |
| 可穿戴设备 | 手环/手表 ECG/EDA 采集、IMU 运动检测 |
| 虚拟康复 | VR/游戏化康复训练、远程康复系统 |
| 运动康复 | 手部康复、骨科康复训练引导 |
